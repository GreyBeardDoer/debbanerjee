I like a minimal online editor for writing online. It should not come in my way of putting my thoughts into words when I am writing.

Yet, it should provide all the required editing and formatting tools that are expected from a modern online writing platform.

IMHO, Ghost has the best editor for online writing at present.

Today, I'll list the steps to install a local version of Ghost on an Ubuntu desktop.

Installing Ghost locally on a Ubuntu VM

  • sudo apt-get upgrade
  • sudo apt-get update
  • Install cURL - sudo apt install curl
  • install NGINX - sudo apt-get install nginx
  • Allow NGINX through firewall - sudo ufw allow 'Nginx Full’
  • Install MySQL - sudo apt-get install mysql-server
  • Install supported version of Node JS - curl -sL [https://deb.nodesource.com/setup_16.x](https://deb.nodesource.com/setup_16.x) | sudo -E bash and sudo apt-get install -y nodejs
  • Instal Ghost CLI - sudo npm install ghost-cli@latest -g
  • Test with - ghost help
  • Create the directory where Ghost will be installed locally - mkdir ghost-local
  • Change to that directory - cd ghost-local
  • Install Ghost locally - ghost install local
  • Once the installation is complete, run and stop Ghost with - ghost start and ghost stop commands.
  • Now you can access your local Ghost installation by visiting the URL - "https://debbanerjee.com/blog" from you browser.
